

1. 변수란
  - 하나의 값을 저장할 컴퓨터 상의 하나의 메모리 공간
  - 어떤 값을 이름으로 가리킬 때 사용하는 기능
  - 기본 형식
    let age = 25; -> age를 변수라고 한다.
  - '변수를 초기화 한다'고 한다.
  - 변수값은 변경할 수 있기 때문에 반드시 초기화할 필요는 없다.
  - 값을 가리키는 이름이면서 동시에 저장소이다.
  - 저장한 값은 프로그램을 실행하는 도중에 변경 가능
  - var 키워드로 선언한 변수 역시 let처럼 프로그램 실행 과정에 값 변경 가능
  - var를 이용한 변수 선언은 let과는 다르게 이름을 중복해서 선언해도 실행 가능
  - 같은 이름을 또 쓸 수 있도록 하면, 코드가 많아져 복잡해진다.
  - var을 사용하면 혼란을 야기할 수 있기 때문에 가급적 변수 선언 키워드는 let을 사용

  - 변수 선언
    - 데이터 값을 저장할 수 있는 변수 메모리 하나 생성
    - var 변수명을 같이해서 여러 변수 메모리를 생성할 수 있는 문제점의 보안책으로
      let 키워드 사용해서 변수 메모리를 만들어라고 명시
    (1) 변수 선언하기
      > var currentYear;
      > var birthYear;
      > var age;
    (2) 변수 한꺼번에 선언하기
      > var currentYear, birthYear, age;
    (3) 변수 선언과 값 할당 따로하기
      > var birthYear;
        birthYear = 1995;
    (4) 변수 선언과 값 할당 같이 하기
      > var currentYear = 2021;

    - 변수 메모리에 저장할 수 있는 데이터 종류
      - 문자열, 숫자, 논리형, null, 참조자료형...

    


2. 변수 선언의 규칙
  - 변수를 사용하기 위해서 변수를 구별할 수 있도록 이름을 붙여 준다.
  - 프로그램에서 사용할 변수 이름은 서로 다르게 만들어야 한다.
  (1) 변수 이름은 영어 문자와 언더스코어(_), 숫자를 사용한다.
    > 첫 글자는 영어 대소 문자나 언더스코어(_)만 쓸 수 있으며, 숫자나
      기호, 띄어쓰기는 허용 X
      (예) now, _now, now25, now_25 등은 사용 가능
          25now, now 25, *now 는 사용할 수 없음

  (2) 영어 대소 문자를 구별하며 예약어는 변수 이름으로 사용 X
    > number, Number, NumBer 모두 다른 변수 이름으로 인식
    > var, let과 같이 미리 정해 놓은 예약어는 변수 이름으로 사용 X

  (3) 여러 단어를 연결한 변수 이름은 중간에 대문자를 섞어 쓴다.
    > 한 단어로 이루어진 변수 이름은 모두 소문자, 두 단어 이상의 변수 이름은
      totalArea 또는 TotalArea 또는 Total_Area처럼 중간에 대문자를 섞어 사용.
      -> 낙타 표기법(camel case)라고 한다.

  (4) 변수 이름은 의미 있게 작성
    > 변수 이름만 보고도 어떤 값인지 추측할 수 있도록 선언하는 것이 좋다.



3. 상수
  - 상수는 변수와 다르게 프로그램 실행 과정에서 값을 변경할 수 없다.
  - 사용 방법
    const age = 25;
  - 상수는 선언과 동시에 값을 할당하는 초기화 과정이 필수
  - 하나의 값을 저장하되 값을 변경해서 저장할 수 없는 하나의 메모리공간
  - 상수 이름을 지을 때 상수 이름에 포함될 수 있는 기호
    > _ 와 $ 만 허용, 그 이외의 기호는 사용 X
