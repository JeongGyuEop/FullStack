


1. 자바스크립트 작성방법 2가지
  (1) html 파일 내부에 내부 자바스크립트 사용하기
    - 자바스크립트 소스 코드가 짧을 경우 웹 문서에서 자바스크립트를
      실행할 위치에 바로 코드를 작성
    - <script></script> 사이에 실행할 자바스크립트 소스를 작성
    - <script></script> 태그는 웹 문서 안에 어디든 위치 가능
    - <script></script> 태그는 하나의 문서에 여러 개 사용 가능
    - 웹 문서에서 이미지나 텍스트 등의 요소를 제어하는 경우가 많으므로,
      되도록이면 이미지나 텍스트 등을 다 표시한 후에 실행하는 것이 좋다.
        > <body></body> 태그 직전에 자바스크립트 소스를 삽입하는 이유
    - 영어 대소 문자를 구별하기 때문에 소스코드 작성 시 주의
    - 변수 이름이나 함수를 지정할 때는 영문 대소 문자를 정확하게 구별해야 한다.


  (2) html 파일 외부에 .js파일에 만들어 놓고, .html파일로 불러와 사용하는 방법
    - 웹 문서 안에는 직접 자바스크립트 소스가 드러나지 않아 소스코드가 깔끔해진다.
    - <script></script> 태그 없이 자바스크립트 소스만 작성하고 확장자는 *.js
      파일로 저장
    - HTML 문서에서 <script></script> 태그의 src 속성을 이용하여 자바스크립트 파일을
      연결한다.
    - 기본 형식
      <script src="외부 스크립트 파일 경로"></script> 




2. 우베 브라우저에서 스크립트를 해석하는 과정
  (1) <!DOCTYPE html>을 보고 문서가 웹 문서라는 것을 확인하고 <html></html> 태그
      사이의 내용을 HTML 표준에 맞춰 읽는다.
  (2) 웹 문서에서 HTML 태그의 순서와 포함 관계를 확인한다. <head></head>와 <body></body>
      태그 사이에 각각 어떤 태그가 있는지 확인.
  (3) 태그 분석이 끝나면 스타일 정보 분석
  (4) <script></script> 태그를 만나면 웹 브라우저 안에 포함된 자바 스크립트 해석기에게
      스크립트 소스를 넘긴다. 자바스크립트 해석기는 <script></script> 사이의 소스를 해석한다.
  (5) 분석한 HTML과 CSS 정보에 따라 웹 브라우저 화면에 표시
  (6) 웹 브라우저에서 분석한 자바스크립트를 실행해서 결과를 화면에 표시
