

1. JavaScript는 어떤 언어인가?
  - HTML : 웹 페이지 요소의 배치
  - CSS : 웹 페이지 요소의 스타일링
  - JavaScript : 웹 페이지 요소의 동작 정의
    > 웹 페이지에 동적인(움직임이 있는) 기능을 장착할 수 있게 도와준다.
    > 버튼의 클릭, 정보의 입력, 페이지 스크롤, 페이지 이동 등 모든 동작은
      자바스크립트로 구현한 기능

2. 자바스크립트 실행 방식
  - '자바스크립트 엔진'이라는 프로그램이 해석하여 실행(기본적으로 웹에 탑재)
  - Node.js와 같은 별도의 실행 환경을 설치하여 실행
  
  (1) 크롬 웹 브라우저에서 개발자 도구를 실행한다.
  (2) F12 또는 오른쪽 상단의 메뉴 -> 도구 더보기 -> 개발자 도구 선택
  (3) 개발자 도구 창에서 콘솔(Console) 탭 선택
    > 콘솔은 자바스크립트의 동작을 기록(로그)하거나 표시할 목적으로 사용
    > 파란색 화살표(>) 기호 오른쪽에 커서가 깜빡이면서 명령을 대기
    > 이러한 커서를 '프롬프트(Prompt)'라고 한다.
    > 프롬프트에 간단한 자바스크립트 명령을 입력하고 Enter 키를 누른다.
  (4) console.log는 괄호 안의 문자열을 콘솔에 출력하는 명령어
