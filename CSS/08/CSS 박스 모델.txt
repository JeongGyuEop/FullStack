


1. 블록 레벨 요소와 인라인 레벨 요소
  - 블록 레벨 요소 : 태그를 사용해 요소를 삽입했을 때 혼자 한 줄을
                      차지하는 것.
  - 해당 요소의 너비가 100%라는 뜻
  - 블록 레벨 요소의 왼쪽이나 오른쪽에 다른 요소가 올 수 없다.
  - 대표적인 태그 : <h1>, <div>, <p>

  - 인라인 레벨 요소 : 콘텐츠만큼만 영역을 차지하고 나머지 공간에는
                      다른 요소가 올 수 있다.
  - 한 줄에 인라인 레벨 요소를 여러개 표시할 수 있다.
  - 대표적인 태그 : <span>, <img>, <strong>


2. 박스 모델의 기본 구성
  - 스타일 시트에서는 박스 형태인 요소를 박스 모델(box model)요소 라고 한다.
  - 웹 문서 안에 여러 요소를 원하는 위치에 한 줄에 배치할지, 줄을 바꾸어 배치할지,
    요소와 요소 사이의 간격을 어떻게 조절할지 결정할 수 있다.
  - 콘텐츠 영역, 패딩, 테두리, 마진 등의 요소로 구성됨.
      > 패딩(Padding) : 박스와 콘텐츠 사이의 여백
      > 테두리(Border) : 박수의 테두리
      > 마진(Margin) : 여러 박스 모델 사이의 여백


3. 콘텐츠 영역의 크기를 지정하는 width, height 속성
  - 박스 모델에서 콘텐츠 영역의 크기를 지정할 때 너비는 width, 높이는 height속성을 사용
    > <크기>: 너비나 높이의 값을 px이나 em 단위로 지정
    > <백분율>: 박스 모델을 포함하는 부모 요소를 기준으로 너빗, 높잇값을 백분율 %로 지정
    > auto: 박스 모델의 너빗값과 높잇값이 콘텐츠 양에 따라 자동으로 결정.(기본값)
  

4. 박스 모델의 크기를 계산하는 box-sizing 속성
  - width, height 속성은 박스 모델에서 콘텐츠 주변의 여백이나 테두리를 뺀 콘텐츠
    영역의 크기를 가리킴
  - 여러가지 요소를 배치할 때 실제 박스 모델이 차지하는 크기는 콘텐츠 영역 외에도
    콘텐츠 영역 외에도 콘텐츠와 테두리 사이의 여백, 테두리 두께까지 계산
  - 박스 모델의 너비와 높이를 어떻게 결정할 것인지에 따라 border-box와 content-box 선택
    > border-box : 테두리까지 포함해서 너빗값을 지정
    > content-box : 콘텐츠 영역만 너빗값을 지정.(기본값)
  - 요소의 크기를 쉽게 계산하려면 box-sizing 속성을 border-box로 지정하는게 좋다.


5. 박스 모델에 그림자 효과를 주는 box-shadow 속성
  - 이미지 또는 <div>와 같이 전체 영역에 지정하여 넣을 수 있다.
  - 사용법
    > box-shadow : 수평거리 수직거리 흐림정도 번짐정도 색상 insert
  - 속성 값
    > 수평거리 : 그림자가 가로로 얼마나 떨어져 있는가. 양숫값은 요소의 오른쪽,
                음수값은 요소의 왼쪽에 그림자를 만든다.(필수 속성)
    > 수직거리 :그림자가 세로로 얼마나 떨어져 있는가. 양숫값은 요소의 위쪽,
                음수값은 요소의 아래쪽에 그림자를 만든다.(필수 속성)
    > 흐림정도 : 값을 생략하면 0을 기본값으로 하여 진한 그림자를 표시. 값이 커질수록
                부드러운 그림자를 표시. 음수값은 사용 X
    > 번짐정도 : 양숫값을 사용하면 모든 방향으로 그림자가 퍼져서 박스보다 그림자가
                크게 표시. 음수값은 반대.(기본값 0)
    > 색상 : 한 가지만 지정할 수도 있고, 공백으로 구분해서 여러 개 색상 지정 가능.
            (기본값 : 검정)
    > insert : 안쪽 그림자로 그림
