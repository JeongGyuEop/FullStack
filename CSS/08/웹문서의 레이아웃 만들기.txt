


1. 배치 방법 결정하는 display 속성
  - display 속성을 사용하면 블록 레벨 요소와 인라인 레벨 요소를 바꾸어 사용 가능
  - 웹 문서의 내비게이션을 만들면서 메뉴 항목을 가로로 배치할 때 많이 사용
  - 이미지를 표 형태로 배치 가능
  - 속성 값
    > block : 인라인 레벨 요소를 블록 레벨 요소로 만든다.
    > inline : 블록 레벨 요소르 인라인 레벨 요소로 만든다.
    > inline-block : 인라인 레벨 요소와 블록레벨 요소의 속성을 모두 가지며,
                     마진과 패딩을 지정 가능하다.
    > none : 해당 요소를 화면에 표시하지 않는다.


2. 왼쪽이나 오른쪽으로 배치하는 float 속성
  - 이미지를 표시하고 그 주변에 텍스트가 둘러싸도록 할 수 있다.
  - 속성 값
    > left : 해당 요소를 문서의 왼쪽에 배치한다.
    > right : 해당 요소를 문서의 오른쪽에 배치한다.
    > none : 좌우 어느 쪽에도 배치하지 않음.(기본값)


3. float 속성을 해제하는 clear 속성
  - float 속성을 사용해 웹의 요소를 왼족이나 오른쪽에 배치하면 그 다음에 넣는
    다른 요소에도 똑같은 속성이 전달된다.
  - 


참고. display: inline-block과 float: left 속성의 차이
  - display: inline-block은 가로롤 배치하면서도 기본 마진과 패딩을 가짐
  - float: 가로로 배치될 때 요소에 기본 마진과 패딩이 없다.
    > 필요하다면 패딩을 지정해야 한다.
    > clear 속성으로 플로팅을 해제해야 한다.
