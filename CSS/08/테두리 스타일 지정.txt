



1. 박스 모델의 방향 살펴보기
  - 상하좌우 4개의 방향이 있어서 테두리, 마진, 패딩 등을 지정할 때 한번에
    똑같이 지정하거나, 모두 다르게 지정 가능
  - 예약어를 알아두는 것이 좋다.
    > top : 맨 윗부분
    > right : 오른쪽
    > bottom : 아랫부분
    > left : 왼쪽
  - 시계 방향이라는 순서를 기억


2. 테두리 스타일을 지정하는 border-style 속성
  - 기본 값은 none
  - 테두리를 그리려면 가장 먼저 테두리 스타일의 속성 값을 지정해야 한다.
    > none : 테두리가 없다.(기본값)
    > hidden : 테두리를 감춘다. 표에서 border-collapse: collapse인 경우
              다른 테두리도 표시되지 않는다.
    > solid : 실선으로 표시
    > dotted : 점선으로 표시
    > dashed : 짧은 직선으로 표시
    > double : 이중선으로 표시. 두 선 사이의 간격이 border-width 값
    > groove : 테두리를 창에 조각한 것처럼 표시. 입체 느낌


3. 테두리 두께를 지정하는 border-width 속성
  - 두께를 지정할 때는 1px이나 5px 처럼 크기를 직접 입력할 수도 있고,
    thin, medium, thick 같은 예약어 중에서 선택도 가능하다.
  - 만일 테두리 두께를 4방향 모두 다르게 지정하고 싶다면, border-top-width,
    border-bottom-width 처럼 border와 width 사이에 상하좌우 방향을 넣고 하이픈(-)
    으로 연결
  - 사용법
    border-width: 크기 | thin | medium | thick 
      > thin: 웹 브라우저에서 정의한 얇은 두께, 일반적으로 약 1px
      > medium: 웹 브라우저에서 정의한 중간 두께. 일반적으로 약 3px
      > thick: 웹 브라우저에서 정의한 두꺼운 두께. 일반적으로 약 5px


4. 테두리 색상을 지정하는 border-color 속성
  - 박스 모델에서 테두리 색상을 지정할 수 있다.
  - 4개의 방향의 테두리 색상을 한꺼번에 지정할 수도 있고, border-top-color처럼
    border과 color 사이에 테두리 방향을 넣어 주면 하나씩 지정할 수도 있다.


5. 테두리 스타일을 묶어 지정하는 border 속성
  - 테두리 스타일과 두께, 색상을 한꺼번에 표현
  - 4개의 방향의 테두리 스타일을 다르게 지정하고 싶다면, border-top이나 border-right처럼
    속성 이름에 방향을 함께 써서 지정.
  - 스타일이 같다면 간단히 border 속성만 사용. 두께와 색상, 스타일의 속성값 순서는 상관 X


6. 둥근 테두리를 만드는 border-radius 속성
  - 꼭짓점 부분에 원이 있다고 가정해서 둥글게 처리
  - 원의 반지름을 이용하면 둥근 정도를 나타낼 수 있다.
  - 기본 형식
    border-radius: <크기> | <백분율>
      > 크기 : 반지름 크기를 px, em의 단위와 함께 수치로 표시
      > 백분율 : 현재 요소의 크기를 기준으로 비율(%)로 지정
  - 꼭짓점마다 따로 둥글게 처리하기
    > 박스 모델에서 꼭짓점 4개를 모두 다르게 지정하고 싶은 경우 사용
    > border-radius: <가로 반지름> / <세로 반지름>
      > 타원을 만들고 싶은 경우에 사용
