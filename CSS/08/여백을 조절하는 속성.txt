


1. 요소 주변의 여백을 설정하는 margin 속성
  - 요소와 요소 사이의 간격을 조절 가능
  - 기본 형식
    > margin: <크기> | <백분율> | auto
      > 크기 : 너빗값이나 높잇값을 px나 em 같은 단위와 함께 수치로 지정
      > 백분율 : 박스 모델을 포함한 부모 요소를 기준으로 퍼센트(%)로 지정
      > auto : display 속성에서 지정한 값에 맞게 적절한 값을 자동으로 지정
  - 개발자 도구 창을 통해서 마진을 확인할 수 있다.


2. margin 속성을 사용하여 웹 문서를 가운데 정렬하기
  - margin 속성을 사용해 웹 문서의 내용을 화면 중앙에 배치하려면 우선적으로
    배치할 요소의 너빗값이 정해져 있어야 한다.
  - margin-left와 margin-right의 속성값을 auto로 지정
  - CSS는 웹 브라우저 화면의 너비에서 요소 너빗값을 뺀 나무지 영역을 좌우 
    마진으로 자동 계산.


3. 마진 중첩 이해하기
  - 요소를 세로롤 배치할 경우 각 요소의 마진과 마진이 서로 만나면 마진 값이
    큰 쪽으로 겹쳐지는 문제가 생긴다 -> 마진 중첩 또는 마진 상쇄라고 한다.
  - 여러 요소를 세로로 배치할 때 맨 위의 마진과 맨 아래 마진에 비해 중간에 
    이있는 마진이 지나치게 커지는 것을 방지하기 위함.
  - 마진 중첩은 아래 마진과 위 마진이 서로 만날 때 큰 마진 값으로 합쳐지는 것
  - 오른쪽 마진과 왼쪽 마진이 만날 경우에는 중첩되지 않는다.


4. 콘텐츠와 테두리 사이의 여백을 설정하는 padding 속성
  - 콘텐츠 영역과 테두리 사이의 여백
  - 테두리 안쪽의 여백
  - 박스 모델에서 패딩을 지정하는 방법은 마진과 거의 같다.
  - 4개 방향의 마진을 한꺼번에 지정할 수도 있고, 하이픈(-)을 사용하여
    특정 방향에만 지정할 수도 있다.
