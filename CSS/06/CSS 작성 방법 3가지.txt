
1. 내부 스타일 시트 사용
  - HTML 파일 내부에 CSS 코드를 작성하는 방법
  - HTML에서 제공하는 style태그를 사용하여 작성한다.
  - 사용 형식
      <style>
        /* CSS 코드 */
      </style>
  - head 태그 안에 사용하며, head 태그에 CSS를 작성하면 body 태그에 작성된
    내용이 사용자에게 노출되기 전에 CSS를 불러와 빠르게 디자인 적용이 가능한 장점이 있다.
  - 그러나 웹 브라우저에서 HTML 문서를 해석할 때마다 CSS 코드를 매번 다시 읽어서
    성능상으로 좋지 못하다.
  

2. 외부 스타일 시트 사용
  - CSS 코드를 작성하는 별도의 파일을 만들어 HTML문서와 CSS 를 연결한다. 
  - 별도 파일의 확장자는 .css
  - 연결할 때는 link 태그를 사용
  - 사용 형식
    <link rel="stylesheet" href="css파일 경로>

3. 인라인 스타일 사용
  - HTML 태그에서 사용할 수 있는 style 속성에 CSS 코드를 작성한느 방법
  - 태그에 직접 CSS 코드를 작성하기 때문에 CSS 기본 문법 형식에서 선택자 부분이 필요 없다.

4. 정리
  - 실무에서는 대부분 외부 스타일 시트를 사용
  - 외부 스타일 시트 방법은 코드를 유지 보수하기가 편하고 성능적으로 가장 좋다.
  - 인라인 스타일 벙법은 특별한 경우를 제외하고는 실무에서 거의 사용하지 않는다.
