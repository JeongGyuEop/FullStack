

1. 전체 요소에 스타일을 적용하는 전체 선택자
  - 스타일을 문서의 모든 요소에 적용할 때 사용.
  - 사용 형식
    * { 속성: 값; ....}
  - 웹 브라우저의 기본 스타일을 초기화 할 때 자주 사용.
  - margin과 padding
      > 웹 문서 내용을 브라우저 창에 바짝 붙지 않도록 문서 내용과 브라우저 테두리 
        사이에 여백을 두는 것을 마진과 패딩이라 한다.
      > margin과 padding 여백을 0으로 지정하면 웹 문서의 여백을 없애서 이미지가
        웹 브라우저 창에 딱 붙어 나타난다.

  ? 태그와 요소는 어떻게 다른가 ?
    - 태그는 말 그대로 태그 자체를 말한다.
    - 요소는 태그를 적용한 것을 말한다.
    - p 태그를 적용한 스타일 (x), p 요소에 적용하는 스타일 (o)


2. 특정 요소에 스타일을 적용하는 타입 선택자
  - 특정 태그를 사용한 모든 요소에 스타일을 적용
  - 지정한 해당 태그를 사용한 모든 요소에 적용된다.
  - 사용 형식
    태그명 { 스타일 규칙 }


3. 특정 부분에 스타일을 적용하는 클래스 선택자 (중요 !!)
  - 같은 태그라도 일부 다른 특정 부분만 선택해서 스타일을 적용한다.
  - 클래스 이름을 사용해서 다른 선택자와 구별하는데, 클래스 이름 앞에 마침표(.)를
    반드시 붙여야 한다.
  - 사용 형식
    .클래스명 { 스타일 규칙 }
  - 클래스 스타일(클래스 선택자를 사용해 만든 스타일)을 적용할 때는 태그 안에
    class="클래스명" 처럼 class 속성을 사용해 지정한다.
  - 클래스 스타일은 여러 곳에 적용 가능하다.
      class.html파일의 .accent스타일과 <span> 태그
  - 요소 하나에 클래스 스타일을 2개 이상 적용가능하다.
    > 공백으로 구분하여 스타일 이름을 적으면 된다.
      <h1 class="accent bg"></h1>


4. 특정 부분에 스타일을 한 번만 적용할 수 있는 id 선택자
  - 클래스 선택자와 마찬가지로 웹 문서의 특정 부분을 선택해 사용
  - 마침표(.) 대신 #기로를 사용한다.
  - 사용 형식
      #아이디명 { 스타일 규칙 }
  - 클래스 선택자는 문서에서 여러 번 적용할 수 있는 반면,
    id 선택자는 문서에서 한 번만 적용할 수 있다.
  - id 선택자는 중복해서 적용할 수 없기 때문에 문서의 레이아웃과 관련된 스타일을
    지정하거나 웹 요소에 자바스크립트 프로그램을 사용하면서 요소를 구별할 때 사용


5. 같은 스타일 규칙을 사용하는 요소들을 묶어주는 그룹 선택자
  - 여러 선택자에서 같은 스타일 규칙을 사용하는 경우 쉼표(,)로 구분하여
    선택자들을 나열한 후 스타일 규칙을 한 번만 정의한다.
  - 사용 형식
      선택자1, 선택자2 { 스타일 규칙 }
