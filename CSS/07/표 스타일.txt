


1. 표 제목의 위치를 정해 주는 caption-side 속성
  - 표 제목은 <caption></caption> 태그를 사용하여 캡션으로 표시
  - 사용 방법
      caption-side: top | bottom
      > top : 캡션을 표 윗부분에 표시(기본 값)
      > bottom: 캡션을 표 아래에 표시


2. 표에 테두리를 그려주는 border 속성
  - 표 바깥 테두리와 셀 테두리를 각각 지정
  (예) 바깥 테두리는 1px의 검은색 실선, 셀 테두리는 1px의 검은색
        점선으로 표시 -> 표 전체의 바깥 테두리와 셀의 테두리가 모두
        표시되어 화면에서는 테두리가 두 줄 그려진 것처럼 보인다.

3. 셀 사이의 여백을 지정하는 border-spacing 속성
  - 표와 셀에 테두리를 지정하면 셀과 셀 사이에 여백이 조금 생긴다.
  - border-spacing 속성은 셀과 셀 사이의 여백을 조절
  - 사용 방법
      border-spacing: 수평거리 수직거리
  - 수평 거리의 값과 수직 거리의 값을 공백으로 구별해서 나타내며,
    두 값이 같은 경우 1개만 지정해도 된다.

4. 표와 셀 테두리를 합쳐 주는 border-collapse 속성
  - 두 줄로 표시되는 것을 그냥 둘 것인지, 합쳐서 하나로 표시할
    것인지를 결정
  - 이 속성은 <table></table> 태그에 적용되는 스타일에만 지정한다.
    > collapse : 표와 셀의 테두리를 합쳐 하나로 표시
    > separate : 표의 셀 테두리를 따로 표시. (기본값)
