

1. 글자색을 지정하는 color 속성
  - 기본 문법
      color: <색상>
  - color를 사용할 수 있는 속성값은 16진수나 rgb(또는 rgba),
    또는 hsl(또는 hsla) 또는 색상이름
    > 16진수로 표현하는 방법
      - 앞에서부터 두자리씩 묶어 #RRGGBB로 표시한다.
      - RR 자리에(Red), GG 자리에(Green), BB 자리에(Blue)
      - #000000(검은색)부터 #ffffff(흰색) 까지 표현 가능

    > hsl과 hsla로 표현하는 방법
      - hue(색상), saturation(채도), lightness(명도)
      - 0~360도에는 빨간색, 120도 에는 초록색, 240도에는 파란색이
        배치되고 사이에 나머지 색상 배치
      - 채도는 %로 표시, 아무것도 섞이지 않으면 채도가 가장 높은 상태
      - 채도에서 0%은 회색톤, 100%는 원래 색으로 표시.
      - 명도 또한 %로 표시하는데, 0%는 가장 어둡고, 50%는 원래 색,
        100%는 흰색으로 나타난다.
      - 예. hsla(0, 100%, 50%, 0.5)라고 하면 같은 색을 절반쯤 투명하게 만든다.
          > 0.5의 값은 투명도를 나타낸다.

    > 색상을 영문명으로 표현하는 방법
      - red, yellow, black 처럼 색상이름 그대로 사용하기도 한다.

    > rgb와 rgba로 표현하는 방법
      - red, green, blue의 줄임말로 하나도 섞이지 않았으면 0으로 표시하고,
        가득 섞였을 때는 255로 표시
      - rgb 표기법으로 파란색 지정하기
        > h1{ color: rgb(0,0,255); }
      - rgb 표기법으로 불투명도 지정하기
        > h1{ color: rgba(0,0,255,0.5); }

  2. 텍스트를 정렬하는 text-align 속성
    - 왼쪽, 오른쪽, 양쪽, 가운데 정렬 등을 지정 가능하다.
      > start : 줄의 시작 위치에 맞추어 문단 정리
      > end : 줄의 끝 위치에 맞추어 문단 정리
      > left : 왼쪽에 맞추어 문단 정리
      > right : 오른쪽에 맞추어 문단 정리
      > center : 가운데에 맞추어 문단 정리
      > justify : 양쪽에 맞추어 문단 정리
      > match-parent : 부모 요소를 따라 문단 정리

    - 정렬 방법ㅇ르 지정하지 않을 경우 왼족 정렬이 기본
    - 왼쪽 정렬에서는 오른쪽에 여백이 생기지만, 양쪽 정렬에서는
      오른쪽에 여백이 생기지 않는다.


  3. 줄 간격을 조절하는 line-height 속성
    - 줄 간격이 너무 좁거나 넓으면 가독성이 떨어짐
    - 해당 속성은 줄 간격을 원하는 만큼 조절 가능
    - 정확한 단위로 크깃값을 지정
    - 문단의 글자 크기를 기준으로 백분율로 지정 가능
      (예) 글자 크기가 12px인 문단의 줄 간격을 2.0으로 했다면,
            실제 줄 간격은 글자 크기의 2.0배인 24px 이 된다.
    - 사용 방법
      p{ font-size: 12px; line-height: 24px; }
      p{ font-size: 12px; line-height: 2.0; }
      p{ font-size: 12px; line-height: 200%; }
    - 세로 정렬할 때도 유용
    - 글자를 세로로 가운데 정렬하려면 line-height 속성값을
      추가하여, 영역의 높이를 나타내는 height의 속성값과 똑같이
      지정한다.

  4. 텍스트의 줄을 표시하거나 없애 주는 text-decoration 속성
    - 텍스트에 밑줄을 긋거나 취소선을 표시
    - 하이퍼링크에 기본적으로 생기는 줄도 없앨 수 있다.

  5. 텍스트에 그림자 효과를 추가하는 text-shadow 속성
    - 본문에서 자자 사용하면 지저분해 보임
    - 사이트 제목과 같은 강조해야할 글자에 사용하면 눈에 띄는 효과
    - 텍스트에 그림자 효과를 추가해 텍스트를 좀 더 입체감 나게 보여줌
    - 사용 방법
      text-shadow: none | <가로 거리> <세로 거리> <번짐 정도> <색상>
    - 속성 값
      > 가로 거리 : 텍스트부터 그림자까지의 가로 거리로 필수 속성이다.
                    양숫값은 오른쪽, 음숫값은 왼쪽에 그림자를 만든다.
      > 세로 거리 : 그림자의 세로 거리로 필수 속성이다. 양숫값은 글자의 아래,
                    음숫값은 글자의 위쪽에 글자를 만든다.
      > 번짐 정도 : 양숫값을 사용하면 모든 방향으로 그림자가 퍼짐,
                    음숫값은 그림자가 모든 방향으로 축소되어 보임.
                    기본값은 0
      > 색상 : 한가지만 지정할 수도 있고 공백으로 구분해 여러 색상을 지정할
               수도 있다. 기본 값은 현재 글자색


  6. 텍스트의 대소 문자를 변환하는 text-transform 속성
    - 영문자를 표기할 때 대소문자를 원하는 대로 바꿀 수 있다.
    - 텍스트를 대소 문자 또는 전각 문자로 변환
    - 영문자에만 적용
      > capitalize : 첫 번째 글자를 대문자로 변환
      > uppercase : 모든 글자를 대문자로 변환 (많이 사용)
      > lowercase : 모든 글자를 소문자로 변환 (많이 사용)
      > full-width : 가능한 모든 문자를 전각 문자로 변환

  7. 글자 간격을 조절하는 letter-spacing, word-spacing 속성
    - letter-spacing 속성은 글자와 글자 사이의 간격을 조절
    - word-spacing 속성은 단어와 단어 사이의 간격을 조절
    - px, em과 같은 단위나 %로 크기를 조절한다.
