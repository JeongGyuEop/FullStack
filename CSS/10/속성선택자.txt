

1. CSS 고급 선택자 종류 중에서 요소선택[속성명] 속성선택자'
  - HTML 요소 중 특정 속성이 있는 요소를 선택하는 선택자
  - 기본 문법
      요소선택[속성명] { 스타일 정의; }
    (예) a[href] { color: red; }
        > 해석 : a요소 모두 선택하고 그 중에서 href 속성이 작성된 a요소들만 최종 선택
    
    a[href] {
      background: aquamarine;
      border: ;
    }
      > 모든 a요소를  선택하되, href 속성이 부여된 a요소들만 최종선택
      > a -> 모든 a 요소들을 선택해 온다. 
      > a[href] -> 모든 a 요소들을 선택해오고, 
                그중에서도 href 속성이 적힌 a 요소들만 최종 선택

  - 유연한 사용 방법
    > 요소선택[속성명]
    >         [속성명]
    > *[속성명]
[실습. attr-1.html]


2. CSS 고급 선택자 중에서 특정 속성값을 이용하여 선택하는 [속성=속성값] 선택자
  - [속성=속성값] 선택자는 주어진 속성과 속성값이 일치하는 요소를 찾아 스타일을
    지정할 때 사용
  - 대괄호 [] 안에 속성과 속성값을 넣고 그 사이에 '='기호를 표시
  - 기본 문법
      요소선택[속성='속성값'] { 스타일 정의; }
    (예) a[target='_blank'] { 스타일 정의; }

  - 유연한 사용 방법
    > 요소선택[속성='속성값']
    >         [속성='속성값']
    > ... 등등
[실습. attr-2.html]
  

3. CSS 고급 선택자 중에서 특정 속성값이 표함된 속성 요소를 선택하는 [속성~=값] 선택자
- [속성~=값] 선택자는 여러 속성값 중에서 해당 속성값이 포함된 요소를 최종 선택
- 속성이 하나면서 속성값이 여러 개인 경우에 특정 속성값을 찾는 데 편리하다.
  (예) [class ~= button] { 스타일 정의; }
    > class 속성에 대한 값 중에서 button 이라는 단어가 '포함'되어 있는 요소만 최종 선택
    > 대소문자 구별하는 것에 주의
[실습. attr-3.html]


4. CSS 고급 선택자 중에서 특정 속성값이 포함된 요소를 선택하는 [속성|=값] 선택자
  - 특정 속성값이 포함된 요소를 선택하는 선택자 입니다.
  - 속성값은 한 단어로 일치해야 한다.
  - 앞에서 배운 [속성~=값] 선택자는 속성에 대한 값이 하이픈(-)으로 연결한
    단어가 있으면 선택해 오지 않는다.
  - 즉, [속성|=값] 선택자는 지정한 값과 정확하게 일치하거나 지정한 값을 포함해서
    하이픈(-) 으로 연결된 값도 모두 선택해 온다.
[실습. attr-4.html]


5. CSS 고급 선택자 중에서 특정 속성값으로 시작하는 속성 요소를 선택하는 [속성^=값] 선택자
  - 속성 값이 정확하게 일치하지 않더라도 특정 속성값으로 시작하는 요소를 선택하는 선택자
  - 기본 문법
      a[title ^= eng] { 스타일 정의; }
[실습. attr-5.html]


6. CSS 고급 선택자 중에서 특정 값으로 끝나는 속성 요소를 선택하는 [속성$=값] 선택자
  - 속성에 대한 값으로 끝나는 단어가 있는 특정 요소를 선택하는 선택자
  - 기본 문법
      [href $= xls] { 스타일 정의; }
[실습. attr-6.html]


7. CSS 고급 선택자 중 일부 속성값이 일치하는 요소를 선택하는 [속성*=값] 선택자
  - 속성값이 어느 위치에 있든지 지정한 속성 값이 포함되어 있다면 해당 요소를 선택
  - 기본 문법
    (예) href 속성값 중에 'w3'가 포함된 요소를 선택한다면
      [href *= w3] { 스타일 정의; }
[실습. attr-7.html]
