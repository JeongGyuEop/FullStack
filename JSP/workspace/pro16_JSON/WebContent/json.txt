

JSON이란?
	JSON(JavaScript Object Notation)
		- 데이터를 저장하거나 전송할 때 사용하는 간단한 파일 형식
		- JSON 데이터는 키-값 쌍으로 표현하며, 주로 웹에서 서버와 클라이언트 간의
		  데이터 교환에 사용되는 파일 형식이다.
		  
	JSON 특징
		- JSON은 자바스크립트를 확장하여 만들어졌다.
		- JSON은 자바스크립트 객체 표기법을 따른다.
		- JSON은 사람과 기계가 모두 읽기 편하도록 고안되었다.
		- JSON은 프로그래밍 언어와 운영체제에 독립적이다.
		- 비동기 브라우저/서버 통신(Ajax)를 위해 XML 데이터를 
		  대체하는 데이터 전송 형식 중 하나

	JSON의 여러 가지 자료형
		- 숫자형, 문자열, 배열, 객체, null, Boolean

		예1. 키-값 구조 : 데이터는 {} 감싸고, "키":"값" 형식으로 구성한다.
			{"키1":"값1", "키2":값2"} <- JSONObject
			{"name","John", "age":30, "city":"Seoul" } 
			
		예2. 값으로 배열을 넣을 수 있다. 배열 []대괄호로 표시한다.
			{
				"people": [
							{"name":"John", "age":30 },
							{"name":"Jane", "age":25 }				
						  ]
			}
